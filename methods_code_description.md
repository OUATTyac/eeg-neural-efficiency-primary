## EEG preprocessing

EEG data were recorded using the Emotiv EPOC X system and exported in XDF format.
Signals were filtered between 1–40 Hz and re-referenced to the average.

Power spectral density was computed using Welch’s method.

## Feature extraction

Theta (4–8 Hz) and alpha (8–12 Hz) power were extracted from predefined regions of interest.

## Data processing

Participants with missing values were excluded.
Outliers were controlled using winsorization (95th percentile).
Power values were log-transformed (dB).

## Statistical analysis

Paired t-tests and repeated-measures ANOVA were conducted.
Post hoc tests were corrected using Bonferroni adjustment.

## Additional analyses

Spearman correlations were used for SES due to non-normal distributions.
